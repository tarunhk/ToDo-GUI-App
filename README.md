# 📝 Java To-Do List GUI App

## 📌 Project Description
This project is a simple **To-Do List Desktop Application** built using **Java Swing**.  
The application allows users to add, delete, and manage daily tasks through a graphical user interface.

It demonstrates Java GUI development and Object-Oriented Programming concepts using Swing components.

---

## 🎯 Objective
The objective of this project is to:

- Build a desktop GUI application using Java  
- Understand Swing components  
- Implement event handling  
- Manage tasks dynamically  
- Create a user-friendly interface  

---

## ▶️ How to Run the Program

### Step 1: Compile
Open terminal inside project folder and run:
```bash
javac ToDoApp.java
```

### Step 2: Run
```bash
java ToDoApp
```

A GUI window will open.
